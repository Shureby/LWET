# LWET

中文用户可直接跳转到：[LWET 中文说明](#lwet-中文说明)

LWET stands for **Let's Write Everything Together**.

It is a collaborative branching-story platform where readers and writers can build stories together. A story does not need to follow one fixed path: each chapter can continue in different directions, and different writers can create new branches from the same starting point.

## What You Can Do

- Read public stories and explore different story branches.
- Register an account to save your reading progress.
- Create your own stories and submit new chapters.
- Continue an existing chapter or create a parallel branch.
- Use ratings and tags to help readers understand the intended audience and content themes.
- Switch the interface between supported languages.
- Receive moderation feedback when submitted content needs review or revision.

## Who This Is For

LWET is for people who enjoy collaborative writing, branching fiction, shared world-building, multilingual stories, and experimental storytelling formats.

The platform is still under active development. Some features may change as we improve the writing, reading, moderation, and discovery experience.

## Website

- Production: <https://lwet.cc>
- Staging / testing site: <https://staging.lwet.cc>

The staging site may contain experimental features and test data. Its data is separate from the production site, so accounts, stories, reading progress, and submissions on staging do not sync with <https://lwet.cc>.

Regular users may use staging to try new features or reproduce issues safely. Use the production site for normal long-term reading and writing.

## How To Report Bugs Or Suggest Improvements

Please use GitHub Issues in this public repository:

<https://github.com/Shureby/LWET/issues>

When reporting a bug, please include:

- What you were trying to do.
- What actually happened.
- What you expected to happen.
- The page or feature where the problem occurred.
- Your browser and device type if relevant.
- Screenshots or screen recordings if they help explain the issue.

When suggesting a feature, please include:

- The user problem you want to solve.
- The workflow you expect.
- Why the feature would improve the reading or writing experience.

Please do **not** include passwords, private tokens, private emails, payment details, or other sensitive personal information in public issues.

## Repository Note

This public repository is used for user-facing feedback, feature requests, and issue tracking. The private development repository is used for implementation work.

# LWET 中文说明

LWET 是 **Let's Write Everything Together** 的缩写。

这是一个协作式多分支故事平台。故事不必只有一条固定路线：每个章节都可以继续向不同方向发展，不同作者也可以从同一个章节出发，写出新的分支。

## 你可以做什么

- 阅读公开故事，并探索不同的故事分支。
- 注册账号，保存阅读进度。
- 创建自己的故事并提交新章节。
- 续写已有章节，或创建平行分支。
- 使用分级和标签，帮助读者理解故事适合的受众和内容主题。
- 在支持的语言之间切换界面。
- 当提交内容需要审核或修改时，接收审核反馈。

## 这个网站适合谁

LWET 适合喜欢协作写作、分支小说、共同世界观创作、多语言故事和实验性叙事形式的用户。

平台仍在持续开发中。随着我们改进写作、阅读、审核和发现体验，部分功能可能会继续调整。

## 网站地址

- 正式站：<https://lwet.cc>
- 测试站：<https://staging.lwet.cc>

测试站可能包含实验功能和测试数据。测试站和正式站的数据不互通，因此测试站里的账号、故事、阅读进度和提交内容不会同步到 <https://lwet.cc>。

普通用户也可以使用测试站来体验新功能或复现问题。长期阅读和正式创作请使用正式站。

## 如何提交错误或建议

请在这个公开仓库的 GitHub Issues 中提交：

<https://github.com/Shureby/LWET/issues>

提交错误时，请尽量说明：

- 你当时想做什么。
- 实际发生了什么。
- 你原本期待发生什么。
- 问题出现在哪个页面或功能。
- 如果相关，请说明浏览器和设备类型。
- 如果有帮助，可以附上截图或录屏。

提交功能建议时，请尽量说明：

- 你希望解决的用户问题。
- 你期待的使用流程。
- 这个功能为什么能改善阅读或写作体验。

请不要在公开 issue 中填写密码、私密 token、私人邮箱、付款信息或其他敏感个人信息。
